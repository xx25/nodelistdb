About Domain Registrars: This report shows which registrars the domains used by FidoNet nodes are registered with, based on WHOIS data collected by the testing daemon.

Unknown registrar: Many registries (especially ccTLDs and GDPR-redacted registrations) do not publish the registrar name via WHOIS. Those domains are counted as "Unknown / not published".

Nodes: The number of FidoNet nodes using hostnames under the registrar's domains (based on the last 30 days of test data).
